## Overview
Bloomind Garden is a mobile application concept designed to support mental well-being through reflection.
Users record positive memories using text, images, or voice notes, which grow into flowers or trees in a
virtual garden representing emotional growth over time.

## Key Features
- Memory creation using text, images, or voice
- Visual growth of memories as plants
- Calm, private, and non-competitive experience
- Emotion-focused and minimalist design

## Design Process
- User research and personas
- Functional and non-functional requirements
- Low-fidelity and high-fidelity prototypes
- Usability testing and evaluation
- Iterative design improvements based on feedback

## Tools Used
- Canva (high-fidelity prototype)
- User interviews & usability testing
- HCI principles and Nielsen heuristics
